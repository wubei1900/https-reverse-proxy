# 反向代理服务

### 本地机器环境

`/proxy/<name>/` 形式的路径会转发到相应的用户机器上

| 路径                 | 目标地址端口            |
| -------------------- | ----------------------- |
| /proxy/name/    | ip:8080     |


### test 测试环境

| 域名            | 目标地址端口      |
| --------------- | ----------------- |
| www.test.com   | ip:80 |

## 使用方法

绑定需要转发的域名到 `ip` 上

```hosts
ip www.test.com
```
